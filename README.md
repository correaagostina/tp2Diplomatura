# Smoking Prediction

Este proyecto entrena modelos de clasificación para predecir la variable **Smoking** usando datos clínicos. Se evaluaron Decision Tree, Random Forest y XGBoost. El mejor rendimiento lo obtuvo **XGBoost optimizado**, que se utilizó para generar las predicciones finales. Al dataset entregado se le agregó una nueva columna llamada **Smoking** con los valores predichos.
